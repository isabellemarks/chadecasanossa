Como enviar as fotos

As duas fotos do casal precisam ser salvas no mesmo lugar do index.html, com estes nomes exatos: casal-hero.jpg e casal-secundaria.jpg. Em caminhos relativos com ./, a imagem precisa estar na mesma pasta do HTML [web:319][web:324].

1. No GitHub, clique em Add file → Upload files
2. Envie a foto principal renomeada para casal-hero.jpg
3. Envie a segunda foto renomeada para casal-secundaria.jpg
4. Deixe os dois arquivos na mesma tela onde está o index.html

Como dar baixa

Você vai dar baixa manualmente: quando alguém pagar, edita o número restante daquela cota no index.html e publica de novo. Para site estático simples, esse é o caminho mais fácil [web:298][web:300].
